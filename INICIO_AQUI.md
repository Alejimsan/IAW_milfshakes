## 🎯 RESPUESTAS A PREGUNTAS CLAVE

**P: ¿Qué necesita MilfShakes?**
R: Un sitio web DINÁMICO (carrito, pagos, inventario en tiempo real)

**P: ¿Servidor web recomendado?**
R: **Nginx** - Mejor rendimiento (10,000+ conexiones simultáneas vs 500 de Apache)

**P: ¿Protocolo obligatorio?**
R: **HTTPS** - Obligatorio para pagos (PCI DSS) y privacidad (GDPR)

**P: ¿Cómo funciona HTTP?**
R: Cliente → Navegador → DNS → TCP → TLS → Petición HTTP → Servidor procesa → Respuesta HTTP → Renderizado

**P: ¿Cómo proteger el servidor?**
R: Certificados SSL, Firewall, WAF, Backups 3-2-1, Monitoreo 24/7
